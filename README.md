# 操作方法
半角ローマ字で検索可能です。

| 機能 | 方法 |
| --- | --- |
|ウィンドウを前面に出す|Ctrl + Shift + z|
|ファイルパスをランチャーに登録する|ファイルをテキストボックスにD&D</br>もしくは</br>ファイルパスをテキストボックスに入力してEnter| 
|テキストボックスに文字列を入力してEnter※|ファイルを検索して実行する| 
|登録済の全パスを表示する|/all|
|登録済のパスを削除する|該当するパスでDeleteキー|

※Ctrl+Enterでファイルパスの親を起動します。
